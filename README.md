> nextjs学习项目

项目地址： https://nextjs.momobaba.top/

## 功能介绍
1. 用户模板(登录、登出)
2. 商品展示以及购买
3. 订单查看
4. 多语言

## 开发介绍

```bash
#安装依赖
pnpm i

#本地运行
pnpm run dev
```
